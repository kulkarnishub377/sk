VIDES: Vehicle Image Detection System

A production-grade vehicle detection and tracking system designed for building high-quality AI training datasets. VIDES handles multiple RTSP camera streams simultaneously with advanced tracking algorithms to prevent duplicates and sophisticated quality control mechanisms.

Key Features:
  • Multi-camera RTSP stream support with independent processing
  • ByteTrack + Kalman filtering for zero-duplicate tracking
  • ROI-based filtering with custom polygon drawing
  • Quality control: blur detection, brightness check, stationary vehicle filtering
  • Real-time speed estimation and direction detection
  • Custom YOLO model supporting 7 vehicle classes (auto_rickshaw, bike, bus, car, mini_bus, tractor, truck)
  • High-performance multi-threaded architecture
  • Organized dataset generation with camera-wise folder structure
Technologies:
  • Ultralytics YOLO for vehicle detection
  • ByteTrack for multi-object tracking
  • OpenCV for image processing and RTSP streaming
  • Python with multi-threading for optimal performance
  • Kalman filters for trajectory smoothing
  • NumPy for numerical computations
Applications:
  • Traffic monitoring and analysis systems
  • AI training dataset creation
  • Smart city infrastructure
  • ANPR (Automatic Number Plate Recognition) integration
  • Parking management systems
