1. The Core Problem

Manual irrigation is labor-intensive and error-prone. Farmers often over-water, leading to soil nutrient leaching and root rot, or under-water during sudden heatwaves.

The Efficiency Gap

A standard sprinkler system operates on a schedule. JalTantra operates on Demand. If the soil moisture is above 80%, the pump stays off, saving electricity and water.

2. System Architecture

The system is built around the **PIC16F877A** microcontroller, chosen for its robustness in industrial environments. It interfaces with capacitive soil moisture sensors (corrosion-resistant v1.2) and controls a submersible pump via a relay.

3. Embedded Implementation

The logic is written in Embedded C using the **XC8 Compiler**. We use the internal 10-bit ADC to map sensor values to a percentage.

controller.c 
#define _XTAL_FREQ 20000000
#define PUMP_RELAY RC0  

void main() {
    ADC_Init();   // Initialize ADC
    
    while(1) {
        int moisture_val = ADC_Read(0);
        // Map 0-1023 to 0-100%
        int percentage = map(moisture_val, 0, 1023, 0, 100);
        
        if (percentage < 30) {
            PUMP_RELAY = 1; // Turn ON Pump
            LCD_String("Status: Watering");
        } else if (percentage > 80) {
            PUMP_RELAY = 0; // Turn OFF Pump
            LCD_String("Status: Optimal");
        }
        __delay_ms(1000);
    }
}

4. Impact & Awards

We deployed JalTantra in a local garden plot for 3 months. Not only did we reduce water usage, but we also observed a visible improvement in leaf turgidity.

This project was recognized as the "Best Eco-Innovation" at the College Tech Fest 2024, validating the need for simple, effective IoT solutions in agriculture.

5. Future Scope: LoRaWAN

To make this commercially viable for large farms (50+ acres), Wi-Fi is insufficient. The next version of JalTantra will utilize **LoRaWAN** for long-range, low-power communication, sending data to AWS IoT Core for predictive analytics.
