E-Zero: AI & Gamification for E-Waste

A cutting-edge PWA that uses Computer Vision (TensorFlow.js) to value old electronics and rewards users for recycling.

Electronic waste is the fastest-growing waste stream globally. Toxic materials like lead, mercury, and cadmium from old gadgets pose a severe threat to the environment, yet only 17.4% of e-waste is properly recycled.

We built **E-Zero**, a comprehensive ecosystem to solve this. It connects citizens, collection agents, and recycling centers through a seamless, rewarding digital experience.

1. Why E-Zero?

Current recycling systems are opaque. Users don't know where to go, what their device is worth, or if it will actually be recycled. E-Zero fixes this with **transparency** and **incentives**.

2. Key Features

AI Identification

Upload a photo of your old phone, and TensorFlow.js identifies the model and estimates its scrap value instantly.

Smart Map

Real-time locator for the nearest e-waste drop-off bin using Leaflet.js and OpenStreetMap.

3. The Tech Stack

We chose a **MERN Stack** (MongoDB, Express, React, Node.js) architected as a **Progressive Web App (PWA)** to ensure accessibility even in areas with poor connectivity.

4. AI Valuation Logic

Using **TensorFlow.js**, we run the MobileNet model directly in the browser. This ensures user privacy (images aren't sent to a server) and gives instant feedback.

identification.js 
import * as tf from '@tensorflow/tfjs';
import * as mobilenet from '@tensorflow-models/mobilenet';

async function classifyImage(imgElement) {
    // Load the model
    const model = await mobilenet.load();
    
    // Classify
    const predictions = await model.classify(imgElement);
    
    // Check if it's electronic waste
    const eWasteKeywords = ['monitor', 'keyboard', 'phone', 'laptop'];
    const isEWaste = predictions.some(p => 
        eWasteKeywords.some(key => p.className.includes(key))
    );
    
    return isEWaste ? calculateValue(predictions[0]) : 0;
}

5. Gamification Rewards

To drive engagement, we implemented a rewards system. Users earn "Eco-Points" for every kg of waste recycled.

  • Leaderboards: Compete with neighbors to be the "Greenest Citizen".
  • NFT Badges: Unique digital collectibles for milestones (e.g., "100kg Recycled").
  • Redemption: Points can be exchanged for coupons at partner electronics stores.
