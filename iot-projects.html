IoT Projects by Shubham Kulkarni

Published on December 11, 2024 | By Shubham Kulkarni

Shubham Kulkarni is an experienced IoT and Embedded Systems developer who has created innovative Internet of Things solutions. This showcase highlights the key IoT projects that demonstrate the technical expertise and hardware-software integration skills of Shubham Kulkarni.

Smart Irrigation System

The flagship IoT project by Shubham Kulkarni is the award-winning Smart Irrigation System. This innovative solution developed by Shubham Kulkarni integrates IoT sensors, cloud computing, and AI to optimize water usage in agriculture. The system created by Shubham Kulkarni achieved a remarkable 30% reduction in water wastage.

Shubham Kulkarni engineered this IoT solution using Raspberry Pi as the central controller, integrated with soil moisture sensors, weather APIs, and relay modules. The system analyzes real-time data and uses machine learning models trained by Shubham Kulkarni to determine the optimal irrigation schedule.

This project by Shubham Kulkarni won Rank 2 at SPPU Startup Olympiad 2025 and was a key component of the AIR 1 winning submission at Smart India Hackathon 2023.

Line Following Robot

Shubham Kulkarni designed and developed an autonomous line-following robot using Arduino Mega and IR sensors. This robotics project by Shubham Kulkarni demonstrates expertise in embedded systems programming, sensor integration, and real-time control systems.

The robot created by Shubham Kulkarni uses infrared sensors to detect and follow a predefined path with high accuracy. The control algorithm developed by Shubham Kulkarni ensures smooth navigation and quick response to path changes.

PIC Microcontroller Projects

During his internship at Emertxe Information Technologies, Shubham Kulkarni worked extensively with PIC microcontrollers. Shubham Kulkarni engineered a multi-mode architecture for a simulated washing machine using PIC microcontrollers, developing a PICsimlab-based simulation model that improved mode-switching accuracy by 20%.

This project showcases the low-level programming skills of Shubham Kulkarni and his ability to work with embedded C and hardware simulation tools.

IoT Cloud Integration

Shubham Kulkarni has extensive experience in connecting IoT devices to cloud platforms. For the Smart Irrigation System, Shubham Kulkarni integrated AWS IoT Core for device management, enabling remote monitoring and control of the irrigation system.

The cloud architecture designed by Shubham Kulkarni includes real-time data streaming, secure device authentication, and a user-friendly dashboard for farmers to monitor their fields remotely.
